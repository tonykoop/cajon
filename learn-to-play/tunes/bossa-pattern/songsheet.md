# Bossa Nova Pattern — Songsheet

**Tempo:** 120 BPM  **Meter:** 4/4  **Difficulty:** ★★★  
**Tradition:** Brazilian bossa nova (traditional pattern, public domain)

## Full pattern (one bar, 16th-note grid)
```
1  e  +  a  2  e  +  a  3  e  +  a  4  e  +  a
B  S  H  B  S  H  S  H  B  H  S  H  S  B  B  H
(R hands play S; L hands play B/H)
```

## Song structure
- Bars 1–4:   Right-hand clave alone (learn the feel)
- Bars 5–8:   Add left-hand bass
- Bars 9–12:  Full pattern
- Bars 13–16: Full + heel fills on every off-beat

**Practice tip:** The bossa pattern is taught by breaking it into clave → bass → heel layers.
Never rush — the groove lives in the space between strokes.
