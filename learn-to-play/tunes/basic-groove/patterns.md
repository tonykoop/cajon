# Basic 4/4 Rock Groove ★☆☆

Each cell = one eighth note (2 cells = 1 quarter beat).

```
Beat:  1  +  2  +  3  +  4  +
       B  -  -  -  S  -  -  -
```

Repeat 4× then add the off-beat tip:

```
Beat:  1  +  2  +  3  +  4  +
       B  -  T  -  S  -  T  -
```

**Form:** 8-bar loop × 4 (32 bars total). Play at 80 BPM to start.

**Practice tip:** Lock the bass on beat 1 first. Add slap on 3. Add tips last.
