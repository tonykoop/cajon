# Waltz Groove ★☆☆

3/4 time: three beats per measure. Bass on 1, slap on 2 and 3.

```
Beat:  1  2  3
       B  S  S
```

Variation with tip on the off-beat:
```
Beat:  1  +  2  +  3  +
       B  -  S  T  S  T
```

**Form:** 12-bar loop. Tempo: 60–90 BPM.

**Practice tip:** Count "1-2-3, 1-2-3" aloud. The bass on "1" is the anchor.
