# Flamenco Rumba — Songsheet

**Tempo:** 110 BPM  **Meter:** 4/4  **Difficulty:** ★★☆
**Tradition:** Afro-Cuban/Flamenco (traditional, public domain)

## Clave pattern (right hand)
```
S - - S - - S - | - - S - S - - -
```

## Full pattern (8 bars)
Right (slap clave) over left (bass quarter notes). Count 8 bars, then cycle.

**Practice tip:** Master the clave alone for 5 minutes before combining hands.
