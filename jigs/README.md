# Cajon Jigs

The cajon looks like a cabinet until the tapa goes on. These jigs keep it an instrument.

## Required for CJ-S first prototype

1. **Square assembly fixture** - flat baseboard with fixed 90-degree fences and movable clamp blocks. Use during dry fit and glue-up.
2. **Tapa screw template** - hardboard or acrylic drill template indexed from the bottom outside edge and centerline. This protects the screw spacing and the unscrewed slap-zone margin.
3. **Back-panel template** - sound-hole center, tuner holes, and snare mount pilot layout. This can be combined with the tapa template if made from clear acrylic.

## Optional variant jigs

- **Finger-joint sled** for a table-saw V1 build.
- **Spline sled** for V4 miter+spline corners.
- **CNC dovetail fixture** for V2 half-blind dovetails.

## Build order

Make the square assembly fixture and screw template before cutting final tapa stock. The templates should be tested on scrap with the exact screws and tuner hardware from `bom.csv`.
