# Cajon Images

This folder is reserved for real CJ-S build photos, validated renders, and
future concept/support imagery.

Generated images and site previews are concept/support only. They must not be
used to infer panel thickness, snare contact geometry, screw positions, joinery
tolerances, or CNC toolpaths. Record any accepted visual artifact in
`../visual-output-register.csv` before referencing it from the README, explorer,
site, print packet, or capstone deck.

Recommended future evidence:

- `cj-s-hero.jpg` after the first real prototype is built.
- `snare-contact-closeup.jpg` after snare preload is set.
- `tapa-screw-pattern.jpg` after the tapa is mounted.
- `validation-setup.jpg` showing tap-test or spectrum capture setup.
